# EX-P-F ⚛️ Mini SPA en React con API Pública

## Descripción
Este proyecto es un **examen final** de los cursos realizados.  
Se trata de una **Mini SPA** desarrollada en **React**, que consume una API pública.

Incluye:  
- **Home:** Hero y CardList de películas.  
- **Entities:** Listado completo con `fetch` y paginación.  
- **Contact:** Formulario con validación y mensaje de éxito.  
- **Estado global:** Usando **Zustand**.  
- **Rutas y Layouts:** Con **React Router**.  
- **Diseño:** Con **Bootstrap** y **Tailwind CSS**.
